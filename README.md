# Xzq Notes Documentation

This is the documentation website for [xzq notes][].

## Development

```bash
// install dependencies
pnpm install
// run only dev server
pnpm dev
```

## Clear Cache

If you upgrade deps about astro, you may need to clear the cache:

```bash
rimraf .astro
rimraf node_modules
pnpm install
```
